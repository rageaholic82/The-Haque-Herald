### 1. Adding a New Post
* Go to the `posts/` folder.
* Create a new file ending in `.md` (e.g., `january.md`).
* The website will automatically put the **newest** post at the top.

### 2. Adding Images
* Upload your photo to the `images/` folder.
* In your post, type: `![Description](images/your-photo.jpg)`

### 3. Adding Video
* Upload an `.mp4` to the `images/` folder.
* Paste this into your post:
  <video width="100%" controls><source src="images/video.mp4" type="video/mp4"></video>

### 4. Formatting Tips
* `# Title` (Big)
* `## Subtitle` (Medium)
* `**Text**` (Bold)
* `* Item` (Bullet point)

### 5. Editing Made Easy
* Press the **period key (.)** on the main repo page to open the visual editor.
* Use `Ctrl + Shift + V` to preview your post while writing.
